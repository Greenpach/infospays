#  Countries Explorer

Application web développée avec React.js permettant d'afficher et filtrer les informations des pays du monde via l’API RestCountries.

---

##  Fonctionnalités

- Récupération des données depuis l’API RestCountries
- Affichage des pays sous forme de cards
- Affichage du drapeau, nom, capitale et population
- Filtrage dynamique
- Tri par population
- Interface réactive avec React Hooks (useState, useEffect)

---

##  Stack technique

- React JS
- Axios
- JavaScript ES6+
- CSS
- API RestCountries

---

##  Installation

Clone le projet :

bash
git clone https://github.com/Greenpach/infospays

Installe les dépendances :

npm install

Lance le serveur :

npm start
